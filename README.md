# Henson

Managing your Puppets.

![](http://userserve-ak.last.fm/serve/_/83451/Jim+Henson.jpg)

## Installation

Add this line to your application's Gemfile:

    gem 'henson'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install henson

## Usage

```
henson install
henson update
henson package
henson clean
```

## Developing
